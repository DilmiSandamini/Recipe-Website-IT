# FLAVORIZ - Recipe Website

Flavoriz is a modern, responsive recipe website designed to provide users with a delightful cooking experience. It features various categories of food, including Rice & Curry, Noodles, Soups, and Desserts.

## 🚀 Features
- **Responsive Design:** Optimized for desktops, tablets, and mobile devices using CSS Media Queries.
- **Interactive Navigation:** Smooth navigation with a custom burger menu for mobile view.
- **Detailed Recipe Pages:** Each recipe includes a clear ingredient list and step-by-step instructions.
- **Modern UI:** Clean aesthetic with hover effects and intuitive layout.

## 🛠️ Technologies Used
- **Frontend:** HTML5, CSS3 (Flexbox, Grid, Media Queries)
- **Interactivity:** JavaScript (for menu navigation)
- **Icons & Images:** Asset-based icons and high-quality food photography.

## 📂 Project Structure
- `index.html`: Home page containing all categories.
- `pages/`: Detailed pages for individual recipes (e.g., Beef Curry, Chicken Noodle Soup).
- `css/`: Modular stylesheets for layout, header, footer, and recipe details.
- `js/`: Script files for UI interactivity.
